Exercise 7:

1. Why it is better than letting every teacher write their own style.

Because of the hierarchical structure of style sheets, the school should let the teacher be able to edit their own style. For example, the school could provide the general style which apply for all the professor's website by using the imported style sheets (External CSS), then each professor could change their decoration by using embedded style sheets or in-line style sheets (Internal CSS or In-line CSS)

Write a list of ten or more HTML tags and classes that may be used for the website

  1. html tag: played as the container for all other html elements
  2. body tag: used to contain a web page's content which include the school logo, link to assignment etc.
  3. h1, h2, h3.. tag: shows the heading of the section
  4. p tag: shows announcement or comments
  5. table tag: shows the table for the exams, quizzes, and homework
  6. img tag: shows school log or student, professor picture
  7. button: used for submit any homework or assignment
  8. form tag: student can fill in the form for survey or feedback
  9. a tag: link to useful resources.
  10. li tag: list any upcoming homework or project
